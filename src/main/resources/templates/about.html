Under the hood

Persistence

Integrations

  • Mail: Gmail SMTP Support
  • Export: OpenCSV Framework
  • Security: Aes/GCM/NoPadding
  • Hashing: BCrypt for Auth

Military-Grade Encryption

We utilize AES-256 in Galois/Counter Mode (GCM) to encrypt your secrets. Unlike standard encryption, GCM provides both confidentiality and data integrity (authentication).

Initialization Vector (IV)

Every encryption operation generates a unique 12-byte IV using SecureRandom, preventing pattern recognition.

Authentication Tag

A 128-bit authentication tag is computed for every ciphertext. If a single bit is altered, decryption will fail immediately.

NoPadding Logic

GCM is a stream-like cipher mode, allowing us to use NoPadding and avoid padding-related attacks while maintaining exact data length.
